# Student-Teacher-appointment
This App is Developed by usingTechnologies  Html,css,Javascript and Editor VSCODE.
This App is used to take appointment from  teacher.
In this,3 main Modules available 1.Admin 2.Teacher 3.Student.
1.ADMIN  :It contains submodules 1.Add teacher(Name,Department,Subject) 2.Update/Delete Teacher 3.Approve register student.
2.TEACHER:It contains submodules 1.Login 2.Schedule appointment 3.Approve/Cancel Appointment 4.View messages 5.View all appointment 6.logout.
3.STUDENT: It contains submodules 1.Register 2.Login 3.Search teacher 4.Book appointment.
